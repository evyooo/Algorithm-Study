# 알고리즘 스터디

* 스터디 진행 <br>
매주 월요일
과제 7시까지 올리기(깃허브) <br>
7시-7시 10분 이번 주 과제에 대한 이야기 나누기 <br>
7시 10분 -30분 서로 쓴 코드 읽어보기  <br>
7시 30분 - 8시 문제 별로 돌아가면서 피드백 및 자신의 코드 설명하기(30분) <br>

불참 3000원 <br>
지각 2000원 <br>
